# api_curriculum
API Rest de Curriculum de un usuario

Este projecto es un ejemplo de como realizar un API Rest con Django Rest Framework.

Tambien incluye
 * Login
 * Logout
 * Registro
 * Token de usuarios (se genera al momento de realizar el login)
